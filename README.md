# Simulate emergency accident alert system via VANET using SUMO and NS3

This repository contains the final-term project for the **Network Programming course**, focusing on simulating and analyzing **Vehicular Ad-Hoc Networks (VANETs)**.

This repository includes:
- [Docs](https://github.com/Mojinnn/VANET_project/main/Docs): [Slides](https://github.com/Mojinnn/VANET_project/main/Docs/Slide) & [Report](https://github.com/Mojinnn/VANET_project/main/Docs/Report)
- [result](https://github.com/Mojinnn/VANET_project/main/result):
-- [pcap file](https://github.com/Mojinnn/VANET_project/main/result/pcapfile): Contain .pcap file for analyze with Wireshark
-- [tcl file](https://github.com/Mojinnn/VANET_project/main/result/tclfile): Result of SUMO build to get mobility file
-- [xml file](https://github.com/Mojinnn/VANET_project/main/result/xmlfile): Result of code to view in NetAnim
- [Src](https://github.com/Mojinnn/VANET_project/main/Src): Full source code of project
- [SUMO](https://github.com/Mojinnn/VANET_project/main/Docs): All file need to build mobility file via SUMO with the real map
